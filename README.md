# 个人主页

一个简单的静态个人主页，包含简介、技能、项目作品和联系方式等板块。

## 文件结构

```
.
├── index.html      # 主页面
├── styles.css      # 样式文件
├── script.js       # JavaScript 交互
├── vercel.json     # Vercel 部署配置
└── README.md       # 说明文档
```

## 快速开始

### 本地预览

直接在浏览器中打开 `index.html` 即可预览。

### 部署到 Vercel

1. 在 [Vercel](https://vercel.com) 创建账户
2. 安装 Vercel CLI: `npm i -g vercel`
3. 在项目目录运行: `vercel`
4. 按照提示完成部署

或者直接导入 GitHub 仓库到 Vercel 自动部署。

## 自定义

修改以下文件中的内容来定制你的个人主页：

- `index.html`: 更新个人介绍、技能、项目等信息
- `styles.css`: 调整颜色和样式
- `script.js`: 添加或修改交互效果

## 功能特点

- 响应式设计，支持移动端
- 深色主题
- 平滑滚动动画
- 现代简约风格
